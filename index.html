Cara daftar dan upload tugas di git

Create Akun GitHub

  • buka browser dan masuk ke https://github.com
  • Klik tombol "Daftar" di sudut kanan atas untuk membuat akun GitHub baru.
  • Isi informasi yang diperlukan, seperti nama pengguna, alamat email, dan kata sandi. Ikuti petunjuknya untuk menyelesaikan pendaftaran.

Membuat Repositori Tugas

  • Setelah selesai membuat akun, klik tombol "+" di sudut kanan atas dan pilih "Repositori baru".
  • Beri nama Repository sesuai keinginan
  • Pilih visibilitas repository (public atau private) sesuai kebutuhan
  • Centang opsi "Inisialisasi repositori ini dengan README" agar otomatis membuat file README.md.

    • jangan lupa untuk membuat folder pada direktory anda. folder tersebut diberi nama yang sama dengan nama repositori yang telah dibuat kemudian tugas atau file yang ingin di upload disimpan kedalam folder yang telah dibuat.

  • setelah itu masuk ke terminal, masuk pada folder yang telah dibuat lalu jalankan perintah "git init"

    git init adalah perintah yang digunakan untuk membuat repository git kosong.sebelum itu,pastikan aplikasi git telah terinstall di komputer anda

  • setelah itu jalankan perintah "git status" perintah ini digunakan untuk melihat status repositori
  • "git add ." untuk menambahkan semua perubahan yang ada di dalam direktory kerja ke staging area atau tempat di git dimana anda menyiapkan perubahan. atau "git add namafile" untuk menambahkan file tertentu saja.
  • git commit -m "first commit" Perintah git commit digunakan untuk membuat commit baru di dalam repositori Git. Commit adalah cara untuk menyimpan snapshot dari perubahan yang telah Anda buat dalam repositori Anda.
  • "git push origin main" perintah ini digunakan untuk mengirim perubahan (commit) yang telah Anda buat di cabang main lokal Anda ke cabang main di repositori jarak jauh dengan nama origin.
